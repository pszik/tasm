# tasm 

`tasm` is an assembler for the Triangle Abstract Machine.[^1] It provides
an executable called `tasm` which is run from the command line to create a TAM
binary file from an assembly file.

The executable expects two arguments:

- the name of the source file to read assembly instructions from
- the name of the binary file to create

Neither file has any restriction on file extension.

[^1]: D.A. Watt and D.F. Brown, *Programming Language Processors in Java: 
Compilers and Interpreters*. Harlow, Essex: Prentice Hall, 2000.
